Installation Steps (all done in the terminal)

Client Side 
- REACT
    - Install react using <b>npx-create-react-app</b>
    - Install all the package.json dependencies using <b>npm install</b>

Server Side
- Django Dependencies
    - Install Django

- Non Django Dependencies
    - <b> pip install bs4 </b>
    - <b> pip install requests</b>
    - <b> pip install re </b>
